Directions Service ​

The Directions Service calculates routes and directions between two or more locations for different modes of transportation.

A complete and concise overview of all request parameters and return types can be found in the API Playground. Here, however, a few topics are explained in more detail:

  • The different directions aka routing requests and return types
  • How advanced Routing Options can be defined
  • How to use Custom Models to in influence edge weightings during route calculation
  • How Extra Info like road surface, track type, OpenStreetMap way ID or additional Route Attributes can be requested
  • How geometries in directions responses can be decoded
  • How Instruction Types are encoded in the directions response
